# **App Name**: SkinFlex

## Core Features:

- Responsive Design: Mobile-responsive design for seamless use on any device.
- Skin Preview: Interactive skin preview: let users visualize skins on their devices using provided images.
- Smart Model Suggestion: Dynamic phone model suggestion: AI suggests phone models using a tool if the user is unsure.
- Cart Management: Simple cart management to easily add, remove, and review selected skins.
- Payment Integration: Secure Razorpay payment gateway integration for seamless and secure transactions.
- Order History: Order history dashboard to track past purchases
- Admin Panel UI: Admin panel UI to maintain brands, models, and skins.

## Style Guidelines:

- Primary color: Deep Indigo (#4B0082) for a premium and sophisticated feel.
- Background color: Very light lavender (#F0F8FF) to complement the primary color without overwhelming the user.
- Accent color: Electric Purple (#BF00FF) to highlight interactive elements and calls to action.
- Headline font: 'Poppins', sans-serif, for a modern and geometric style.
- Body font: 'PT Sans', sans-serif, providing a clean and readable text.
- Crisp, minimalist icons in a style that complements the modern aesthetic. Use feather icons.
- Clean and structured layout with clear sections for product display, cart management, and checkout.
- Subtle transitions and animations for loading states, button interactions, and page navigation to enhance user experience.
